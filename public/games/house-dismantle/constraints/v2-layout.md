# V2 Layout — 大别山 4 开间 2 层民居约束文档

## 尺寸

- 总宽 16m，总深 9m，总高 ~8m
- 开间(不等宽): 3 / 4.5 / 5 / 3.5m (梢/次/明/梢)
- 进深: 前房 4m + 走廊 1.5m + 后房 3.5m = 9m
- 一层净高 2.6m，二层净高 2.5m，屋顶 28°

## 隔墙

- 纵墙 x = -5, -0.5, +4.5 (开间边界)
- 前走廊墙 z = +0.5 (前房↔走廊)
- 后走廊墙 z = -1.0 (走廊↔后房)
- 墙上门洞宽 0.9m，通道净空 ≥ 0.7m

## 房门约束

- 堂屋前后贯通（前门→屏风→走廊→后门）
- 走廊两侧房门错开（不对门）
- 卫生间门朝北墙，不对走廊不对房门
- 厨房-餐厅通过走廊门洞直连

## 风水

- 大门居中朝南(堂屋 x=2, z=+4.5)
- 厨房东位(开间3后，灶靠右墙)
- 卫生间凶位(开间4后右角，门朝北)
- 屏风半挡穿堂煞(堂屋 z≈-0.5)
- 楼梯不冲大门(开间2后)
