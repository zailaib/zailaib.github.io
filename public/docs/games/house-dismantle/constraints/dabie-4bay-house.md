# 大别山四开间二层民居 — 全屋约束文档

> Designer Agent 6 步流水线 | 全屋重构

## 场地上下文

| 参数 | 值 | 设计影响 |
|------|----|---------|
| 气候 | 大别山——北亚热带湿润，冬冷夏热 | 低层高保温、接地通风、坡顶 28° |
| 地形 | 山区丘陵 | 石基 0.55m 抬高防潮 |
| 朝向 | 坐北朝南 | 前窗朝南、主入口朝南 |
| 东侧 | 上坡马路（噪音源） | 楼梯+厨卫靠东作隔音缓冲 |
| 西侧 | 安静 | 卧室靠西 |
| 地域 | 大别山民居——硬山顶、青瓦、白墙、石基 | MATS: 青灰瓦+白墙+深灰石材 |

## 0. 整体协调性

- [x] 屋顶：硬山双坡顶 28°（tan28°×4.5m=2.4m 起高）
- [x] 材质：白墙 MATS.wall + 青瓦 MATS.roofTile + 深灰石材 MATS.base
- [x] 比例：16m×9m≈144m²，四开间
- [x] 流线：楼梯居中偏东，东西双卫
- [x] 立面：一层 2.6m + 二层 2.5m，石基 0.55m，总高 ~8.0m
- [x] 地基：石基 0.55m，南北穿孔通风道

## 1. 空间定位

```
HOUSE_W = 16m (4 bays × 4m)
HOUSE_D = 9m
WALL_H1 = 2.6m (一层净高)
WALL_H2 = 2.5m (二层净高)
BASE_H  = 0.55m (石基+通风层)
ROOF_H  = ~8.5m (EAVE_H + 2.4m 起高)
EAVE_H  = FLOOR_H + WALL_H1 + WALL_H2 = 0.55 + 2.6 + 2.5 = 5.65m
FLOOR_H = BASE_H = 0.55m
BAND_Y  = FLOOR_H + WALL_H1 = 3.15m
BAY_W   = 4m
```

## 2. 结构约束

- 外墙厚 0.24m，内墙厚 0.15m
- 柱网：前 5 根 + 后 5 根 + 中轴 4 根（14 根），均在墙面交叉点
- 一层净高 2.6m ≥ 1.8m ✓
- 二层净高 2.5m ≥ 1.8m ✓
- 石基带南北穿孔通风道（6 孔北侧 + 6 孔南侧）

## 3. 通道间隙

- 门洞 ≥ 0.8m
- 走廊宽 ≥ 1.0m
- 楼梯宽 0.9m

## 7. 开口通风

### 一层
- 前窗 ×4（每开间 1 个，1.1×1.4m）
- 后窗 ×4（每开间 1 个）
- 前门 ×1（中间开间偏右，1.6×2.3m）
- 后门 ×1（厨房后，通后院）
- 室内门 ×6（老人房×2、卫1、卫2、厨房→餐厅、餐厅→堂屋）

### 二层
- 前窗 ×4（每开间 1 个）
- 后窗 ×4
- 室内门 ×5（主卧、次卧、书房、儿童房×2）
- 走廊通风窗 ×2（前后各一）

### 接地通风
- 北侧石基 6 孔（0.15×0.3m），冷空气进气
- 南侧石基 6 孔，热空气排出
