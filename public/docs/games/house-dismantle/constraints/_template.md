## 1. 空间定位

- **位置**: (x, z) 中心坐标，相对于 houseGroup 原点
- **尺寸**: 宽 × 深 × 高 (m)
- **朝向**: 门/窗朝向 (front=+Z, back=-Z, left=-X, right=+X)
- **与已有部件的位置关系**: 如「紧贴 wallRight 外侧」「位于 crossWall 与 wallFront 之间」

```js
// 示例
{ x: 7.5, z: 2.0, w: 3.0, d: 3.0, h: 2.8 }
```

## 2. 结构约束

- **墙体**: 厚度、高度、是否承重
- **地面/天花板**: 厚度、高度（floor-to-ceiling 净高 ≥ 1.8m）
- **屋顶穿透**: 是否穿透屋顶、与 roof tiles 的间隙 ≥ 0.1m
- **柱子**: 如有柱子，位置是否在墙面交叉点

## 3. 通道间隙

- **门洞**: 宽 ≥ 0.6m (error), ≥ 0.9m (warning)
- **通道**: 家具间通道 ≥ 0.5m
- **与已有开口的距离**: 不与现有门/窗冲突
