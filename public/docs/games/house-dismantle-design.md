# House Dismantle 3D — Design Spec

> 2026-07-24 | 大别山 4 开间 2 层 (16m×9m)

## 项目目标

交互式 3D 建筑模型，通过平移拆解展示民居空间布局。用户点击分类按钮将房屋按层平移，清楚观察每层每个房间。

## 技术架构

```
house-dismantle/
├── app.js (140行)     — Three.js setup, build, animation
├── config.js (160行)   — 尺寸, 材质, 部件注册, 平移偏移量
├── ui.js (248行)       — 图例, 分类按钮, 框选, 事件
├── translate.js (79行) — 拆解/组装逻辑, 依赖解析
├── roof/index.js       — 青瓦屋顶 + 屋架
├── floor1/             — 一层: walls, floor, rooms, openings
├── floor2/             — 二层: walls, floor, rooms, openings
├── base/index.js       — 石基, 水泥柱, 排水明沟, 雨水缸
├── validate/           — 8 条空间检测规则
├── index.html, style.css
```

## 交互

3 个分类按钮，点击即平移，再点复位：

| 按钮 | 方向 | 目的 |
|------|------|------|
| 屋顶 ↗ | +X 20m | 屋顶右移 → 看二层 |
| 二层 ↗ | -X 20m | 二层左移 → 看一层 |
| 一层 ↗ | 原地 | 保留原位 |

部件列表可单独选中 → 按 1 拆解/组装（带级联依赖）。

## 空间布局

```
        前 (+Z, 南)
  ┌──────┬──────┬──────┬──────┐
  │开间1 │开间2 │开间3 │开间4 │  ← 前门(bay3, x=2)
  │老人房│老人房│客厅  │厨房  │
  ├─门洞─┼─门洞─┼─门洞─┼─门洞─┤  z=0 前后隔墙
  │餐厅  │(楼梯)│      │后门  │  ← 后门(bay4, x=6)
  └──走廊门洞(z≈-2)──────────┘
        后 (-Z, 北)
```
二层：主卧/次卧/书房/儿童房×2（镜像一层布局）
